# Pipex tester

## Installation

- In your Pipex project folder :
```
git clone https://github.com/abdelbenamara/Pipex.git tests/
```

## Usage

```
make
```
```
make all
```

## Options

- Vanilla :
```
make re
```
```
make norm
```
```
make test
```
```
make bonus
```

- Valgrind :
```
make valgrind-test
```
```
make valgrind-bonus
```

- Community :
```
make norm community-tests
```
